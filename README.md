# homebrew-tap

Homebrew tap for Manuel Welsch's macOS apps.

```bash
brew tap Manuel-Welsch/tap
```

## Apps

- **[ClipRef](https://github.com/Manuel-Welsch/ClipRef)** — menu-bar app that saves the clipboard to a file and gives you an `@`-path to paste.
  `brew install --cask Manuel-Welsch/tap/clipref`
- **[dd-sweep](https://github.com/Manuel-Welsch/dd-sweep)** — sweeps orphaned Xcode DerivedData build caches whose source folder is gone.
  `brew install Manuel-Welsch/tap/dd-sweep`
