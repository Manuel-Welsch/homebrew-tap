# homebrew-tap

Homebrew tap for Manuel Welsch's macOS apps.

```bash
brew tap Manuel-Welsch/tap
```

## Apps

- **[ClipRef](https://github.com/Manuel-Welsch/ClipRef)** — macOS menu-bar app that saves the clipboard to a file and gives you an `@`-path to paste.
- **[dd-sweep](https://github.com/Manuel-Welsch/dd-sweep)** — sweeps orphaned Xcode DerivedData build caches whose source folder is gone.
